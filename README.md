# EZ Flights Airline Website

## Database population:
  > cd Server
  > node Seeder -i    (To populate database)
  > node Seeder -d    (To delete everything in database)
  
## To run backend:
- Run in terminal:
- cd Server
- npm run dev
  
## To run frontend:
- Run in terminal:
- cd Client/my-app
- npm start
